## What You Can Expect

- Simple installation steps for each component
- Preview and code examples side by side
- Patterns that feel familiar if you already use shadcn/ui on the web
- Building blocks you can copy, customize, and ship quickly

## Getting Started

If this is your first time using AppyKit UI, begin with `Button`, `Text`, and `Card`. Those three components cover most simple screens and give you a good feel for the styling system before you move into inputs, tabs, and more advanced patterns.

## Built For Real App Screens

These components are designed for the parts of a product people touch every day: onboarding flows, settings pages, profile screens, forms, and content layouts. Use them as a starting point, then shape them around your product instead of building everything from scratch.
