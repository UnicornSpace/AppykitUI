## Ways to Import

1. Images can be imaged from the local file system 
```javascript
 <Image
  source={require('./path/to/local/image.jpg')} style={styles.image}
  />
```


2. Images can be imported from the Remote URL

```javascript
 <Image
  source={{ uri: 'https://example.com/image.jpg' }} style={styles.image}
/>
```


## How to use images in React Native Expo
1. Import the Image components from React Native

2. Import the Image components from expo-image


### 1. Import the Image components from React Native:
```javascript
import { Image } from 'react-native';
```
Example of using Image component from React Native:
```javascript
import React from 'react';
import { View, Image, StyleSheet } from 'react-native'; 
const App = () => {
  return (
    <View style={styles.container}>
      <Image
        source={{ uri: 'https://example.com/image.jpg' }}
        style={styles.image}
      />
    </View>
  );
};
const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  image: {
    width: 200,
    height: 200,
    borderRadius: 10,
  },
});
export default App;
```

### 2. Import the Image components from expo-image:
```javascript
import { Image } from 'expo-image';
```
Example of using Image component from expo-image:
```javascript
import React from 'react';
import { View, StyleSheet } from 'react-native';
import { Image } from 'expo-image';
const App = () => {
  return (
    <View style={styles.container}>
      <Image
        source={{ uri: 'https://example.com/image.jpg' }}
        style={styles.image}
      />
    </View>
  );
};
const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  image: {
    width: 200,
    height: 200,
    borderRadius: 10,
  },
});
export default App;
```
